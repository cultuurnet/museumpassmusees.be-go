Fijn dat je een museumpas hebt!

Vanaf september kan je via de url museumpassmusees.be/go je pas registreren. Voorlopig hoef je dit dus nog niet te doen. Je kan nu al de deelnemende musea bezoeken op vertoon van je pas aan de museumbalie.
Vul je e-mailadres in als je op de hoogte wil blijven van de lancering.

Chouette que vous ayez un pass musées !

A partir de septembre, vous pouvez enregistrer votre pass via l’url museumpassmusees.be/go.Il ne faut donc rien faire à présent. Vous pouvez déjà visiter les musées participants en vous présentant avec votre pass à l'accueil du musée.
Laissez-nous votre adresse e-mail si vous souhaitez être tenu au courant du lancement.
